=======
Install
=======

Fable Elmish Reaction is available as a `NuGet
<https://www.nuget.org/packages/Fable.Elmish.Reaction/>`_ package for ``NETStandard
2.0``.

Paket
=====

.. code:: bash

    > paket add Fable.Elmish.Reaction

NuGet
=====

.. code:: bash

    >  dotnet add package Fable.Elmish.Reaction
